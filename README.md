# Expense Analysis Project

## Description
This project analyzes personal expense data from a messy CSV file.
The data is cleaned, processed, and analyzed using Python and Pandas.

## Features
- Handles missing and invalid data
- Normalizes categories
- Calculates total spending per category
- Identifies highest spending areas
- Visualizes results using charts

## Files
- raw_expenses.csv → original messy data
- clean_expenses_final.csv → cleaned data
- expense_summary.csv → category totals
- Expense_analysis_Project.py → main analysis script

## Skills Used
- Python
- Pandas
- Data Cleaning
- Data Analysis
- Data Visualization

## Outcome
Clear insights into spending behavior and areas of high cost.
